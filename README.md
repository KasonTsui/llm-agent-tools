# LLM Agent Tools

This repository is designed to manage and organize various resources for LLM agents, including prompts, instructions, personas, and skills. It serves as a centralized hub for configuring and enhancing AI developer tools.

## Project Structure

The repository is organized into the following directories under `.github`:

- **prompts/**: Contains task-specific prompts (`.prompt.md`) to guide the AI for particular activities.
- **instructions/**: Stores coding standards, guidelines, and best practices (`.instructions.md`) to ensure consistency.
- **agents/**: Defines AI personas and specialized modes (`.agent.md`) tailored for specific roles or tasks.
- **collections/**: houses curated collections of related items (`.collection.yml`) for easy access and organization.
- **scripts/**: Includes utility scripts for maintenance and automation of repository tasks.
- **skills/**: Contains definitions of AI capabilities (`.skill`) that enable specialized tasks.

## Usage

This structure allows for a modular and scalable approach to managing AI context and capabilities. Each directory serves a specific purpose in shaping the behavior and knowledge of the LLM agent.
